# 🌸 Iris Dataset Analysis

## 📌 Objective
The objective of this task is to explore, analyze, and visualize the Iris dataset using Python libraries such as pandas, matplotlib, and seaborn. The goal is to understand data structure, relationships, and patterns through visualizations.

---

## 🛠️ Approach

1.   Loading the Data
   - Loaded the Iris dataset using seaborn.

2.   Exploring the Data
   - Checked dataset shape, columns, and preview using `.shape`, `.columns`, `.head()`
   - Generated summary statistics using `.describe()` and `.info()`

3.   Data Visualization
   - Scatter Plot: Analyzed relationships between sepal length and petal length.
   - Histogram: Observed distribution of sepal length.
   - Box Plot: Identified spread and outliers in petal length across species.



 📊 Results & Insights

- Clear clustering of species observed in scatter plot.
- Setosa species shows distinct characteristics compared to others.
- Petal length is a strong differentiating feature.
- Minimal outliers detected in the dataset.
