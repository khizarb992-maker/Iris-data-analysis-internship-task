Credit Risk Prediction

Project Overview:

This project aims to the prediction of an loan applicant is going to be default or not by seeing his historical data. It will help the company to make informed decision by identifying potential risks in advance.

In this project we have clean the data, and explore it with data analysis and machine learning  model to see or classify is the loan applicant are likely to default or not.

Dataset:
The dataset was used in this project is the Loan Prediction Dataset

The information about loan applicants are:

Applicant Income
Coapplicant Income
Loan Amount
Loan Term
Credit History
Education
Gender, Marital Status, Dependents, and Self Employment status
Loan Status (Target Variable)

Objective

Predict whether a loan will be approved or Not.
Provide insights of key factors for approval of Loan.
Build and evaluate machine learning models for binary classification.

Steps and Approach

1. Data Cleaning

Handled missing values using mean (numerical) and mode (categorical).
Converted categorical features into numerical values using Label Encoding.
Ensured the dataset is ready for machine learning models.

2. Exploratory Data Analysis (EDA)

Visualized key features like Loan Amount, Applicant Income, and Education.
Identified trends and patterns affecting loan status.
Examined relationships between applicant attributes and loan approval.

3. Model Training

Split data into training and testing sets.
Trained Logistic Regression and Decision Tree Classifier.
Compared model performance using accuracy and confusion matrices.

4. Model Evaluation

Evaluated models based on accuracy and confusion matrix.
Logistic Regression achieved strong predictive performance with clear interpretability.
Decision Tree provided insights into feature importance and decision rules.

Key Insights

Applicants with higher income and strong credit history are more likely to get their loans approved.
Education level plays a role in loan default risk.
Logistic Regression provided a more stable prediction compared to Decision Tree for this dataset.

Skills and Tools Used

Python (Pandas, NumPy, Matplotlib, Seaborn)
Data Cleaning and Handling Missing Values
Exploratory Data Analysis (EDA)
Binary Classification (Logistic Regression, Decision Tree)
Model Evaluation (Accuracy, Confusion Matrix)
